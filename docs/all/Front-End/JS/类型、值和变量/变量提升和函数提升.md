函数提升：

“函数是一等公民”

非匿名函数的声明，会被提升至函数作用域/全局作用域的顶端。

// 指 function boo() {} 这样的形式。我也可以理解为“声明和初始化都提升到顶部”，但是好像正式的说法里没有函数的初始化。

// 可以形象地理解为整个function boo() {} 定义段全部都提升到顶部了。
